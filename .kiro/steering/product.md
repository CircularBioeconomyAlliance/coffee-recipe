# Product Overview

## CBA Indicator Selection Assistant

A specialized chatbot for the Circular Bioeconomy Alliance (CBA) that helps users discover and select sustainability indicators from the CBA Knowledge Base. The application serves as an intelligent assistant for researchers, project managers, and sustainability professionals working on circular bioeconomy initiatives.

## Core Purpose

- **Indicator Discovery**: Help users find relevant sustainability indicators based on project requirements
- **Method Guidance**: Provide detailed measurement methodologies for selected indicators  
- **Context-Aware Recommendations**: Consider location, project type, budget, and technical capacity
- **Knowledge Base Integration**: Leverage AWS Bedrock Knowledge Base as the authoritative source for all indicator data

## Key Features

- **Dual Interface**: Both CLI agent and Streamlit web UI for different user preferences
- **Session Management**: Persistent chat sessions with auto-generated titles
- **File Upload Support**: Process PDF, Excel, CSV, and text files for project context
- **Climate Data Integration**: Fetch relevant weather data using Open-Meteo API
- **CBA Branding**: Nature-focused design reflecting the organization's mission

## Target Users

- Sustainability researchers and consultants
- Project managers implementing circular bioeconomy initiatives
- Organizations measuring environmental and social impact
- Policy makers and development practitioners
